Kittenbot OS

Tired Of Only Having 1 Game On Your Microsoft Makecode Device?

‎

Kittenbot OS Is A Open Source "Operating System" That Allows You To Store Multiple Games On Any Device Designed For Makecode Arcade Games! It Comes With A Calculator, Stopwatch, And Screensaver App, A Settings Menu To Change The Color Scheme And 3 Pre-Made Games (flappy bird, spaceships and ballistic balloons). There Is Also A Editable Version Without Games That You Can Add Your Own Games To! 

‎

To Make Your Own Game You Can Download  The Editable Project Without Games And Add Your Own Code Into One Of The Save Slots!

‎

[Important Note] Optimised for the Meow Bit and may lag much more on other devices (it still lags a bit on the meow bit). To run this on another device download either one of the editable projects, import them to Microsoft Makecode and download the UF2 for your device!
